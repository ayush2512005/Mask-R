Right-click the image above → Mask & Save with Redact.

3. Download-interception prompt

These links have download or target="_blank", so clicking them should pop the bottom-right “Sensitive file detected” card.
  • Download PDF (download attr)
  • Open image in new tab (target=_blank)

On that card: Redact Now opens the editor, Save Anyway proceeds, Don't ask for this site suppresses this hostname (then check the popup).
